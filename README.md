# Diamond Kata

## Written in Rust 

I've used `trycmd` for integration tests and `clap` as an arguments parser.

```sh
$ cargo test
$ cargo run <LETTER>
$ cargo run -- --help
diamond 0.1.0

USAGE:
    diamond <LETTER>

ARGS:
    <LETTER>

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information

```
